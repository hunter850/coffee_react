### Path 組件使用說明

```js
<Path pathObj={{path:['．課程資訊','．愛心拉花']}}/>
<Path pathObj={{path:['．{頁面名稱}','．{頁面名稱}']}}/>
```

```js
// 組件範例
<Btn
    style={{
        backgroundColor: '#B79973', // 按鈕顏色
        color: '#fff', // 字體顏色
        txt: '報名課程', // 按鈕名稱
        width: 100, // 按鈕寬度
    }}
/>

// 需要的屬性照著範例輸入,不需要的空著即可
```
